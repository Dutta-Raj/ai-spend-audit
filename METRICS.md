# Metrics Dashboard

## North Star Metric
**Number of audits with identified savings > $100/month**

Why: This is the sweet spot. Small savings (<$100) likely won't convert. Large savings indicate real Credex opportunity.

## 3 Input Metrics Driving North Star

| Metric | Target | How to Improve |
|--------|--------|----------------|
| **Audit Completion Rate** | >60% | Reduce form friction, autosave, pre-populate tools |
| **Savings Detection Rate** | >25% | Add more audit rules, expand tool coverage |
| **Share Rate** | >15% | Better OG images, twitter cards, share CTAs |

## What I'd Instrument First

1. **Funnel tracking:**
   - Page view → Form start → Form complete → Audit run → Email capture

2. **Saving thresholds:**
   - % of audits with $0, $1-100, $101-500, $500+ savings

3. **Tool-specific overpayments:**
   - Which tools generate most savings (Cursor? Claude?)

## Pivot Trigger
**If <8% of audits show >$100 savings after 1,000 audits** → tool isn't finding enough value. Pivot to:
- Add more tools (Gemini, Windsurf, Perplexity)
- Add "benchmark mode" (compare vs similar startups)
- Add subscription tracking (find unused seats)
